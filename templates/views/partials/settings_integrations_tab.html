Nextcloud

WebDAV integration for EDMS file mirror (Archive / Correspondence / Comm Items).
Mount: Files stored via local mount point or UNC path. Requires NEXTCLOUD_LOCAL_MOUNT_ROOT.
WebDAV: Files uploaded/downloaded directly via WebDAV API. No mount needed - uses Base URL, Username, and Root Path only.
Shared remote root in Nextcloud (WebDAV). Archive and Correspondence folders are created under this root automatically.
Used by folder picker to map Nextcloud folders to local/UNC paths. Required in Mount mode. Optional in WebDAV mode. ENV override: NEXTCLOUD_LOCAL_MOUNT_ROOT.
warning
This mode is insecure and should be used only for internal/testing servers.
